# Gyro Shop Dining

A responsive restaurant website built with Next.js, TypeScript, and Tailwind CSS.

## Run locally

```bash
npm install
npm run dev
```

Then open [http://localhost:3000](http://localhost:3000).

For a production check:

```bash
npm run build
npm start
```

## Business information

- Update phone, address, directions, Instagram, and hours in `data/site.ts`.
- Update menu items, descriptions, prices, and images in `data/menu.ts`.
- The Google Maps embed is configured in `app/page.tsx`.
